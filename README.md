# Менеджер личных финансов

## Описание системы
Веб-приложение «Менеджер личных финансов» предназначено для автоматизации процессов учета, контроля и анализа личных доходов и расходов пользователя. Система позволяет упростить ведение личного бюджета за счет интуитивно понятного интерфейса и предоставляет наглядную финансовую аналитику (графики и диаграммы) для оптимизации трат.

## Технологический стек
Для обеспечения соответствия технического обеспечения выбранным инструментам разработки используется следующий стек технологий:

* **Frontend:** React.js + TypeScript + Vite (обеспечивает высокую скорость работы интерфейса и строгую типизацию).
* **Backend:** Node.js + Express.js (обеспечивает высокую пропускную способность и быстрый отклик сервера).
* **База данных:** PostgreSQL (реляционная СУБД для надежного хранения структурированных финансовых данных).
* **ORM:** Prisma (используется для безопасного взаимодействия с базой данных и исключения ошибок в SQL-запросах).
* **Контейнеризация:** Docker + Nginx (для унификации процесса развертывания и обеспечения отказоустойчивости системы).
